# Enterprise-Networking-Project
Enterprise Network Design using VLANs, Inter-VLAN Routing, DHCP, OSPF, WAN Connectivity, DNS, Web, Email Services, and ACL Security in Cisco Packet Tracer.

## Overview
This project demonstrates an enterprise network built in Cisco Packet Tracer.

## Technologies Used
- VLAN
- Inter-VLAN Routing
- DHCP
- Static Routing
- OSPF
- WAN Connectivity
- DNS Server
- Web Server
- Email Server
- ACL Security

## Network Features

### VLAN Segmentation
- VLAN 10 – HR
- VLAN 20 – Finance
- VLAN 30 – Sales

### Routing
- Static Routing
- OSPF Dynamic Routing

### Services
- DHCP
- DNS
- HTTP/HTTPS
- Email

### Security
- Standard ACL
- Extended ACL

## Objectives
- Improve network scalability
- Enhance security using ACLs
- Automate IP assignment using DHCP
- Enable communication between multiple sites
